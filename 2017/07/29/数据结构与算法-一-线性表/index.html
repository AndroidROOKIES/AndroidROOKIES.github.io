数据结构与算法(一)

发表于 2017-07-29 | 分类于 Java

物理结构（存储结构）

1.顺序存储结构(Arraylist)

优点：查找时效率高

缺点：插入和删除时元素之间要进行相应的改变，所以效率比较低

2.链式存储结构（单线连接）(Linkedlist)

线性表的链式存储结构的特点是用一组任意的存储单元存储线性表的数据元素，这组存储单元可以是连续的，也可以是不连续的。

每个节点（p）包含两个部分：数据域(ai)和指针域(空白处)，前面节点中的指针域存放的是后面一个节点的地址。
在Java中可以把每个节点看成一个类。而指针就是类的引用

用Java代码表示每个节点：

public class P{
    Object data;
    //指针，下个元素的引用(下个元素也是P类型的)
    P next;
}

P p1 = new P();
p1.data = "Isen";
P p2 = new P();
p1.next = p2;

优点：删除和插入的效率高

缺点：查询效率低
