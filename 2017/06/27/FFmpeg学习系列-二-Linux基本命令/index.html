FFmpeg学习系列(二)Linux基本命令

发表于 2017-06-27 | 分类于 FFmpeg

Linux相关命令

清空命令行：reset

文件操作：

显示列表

ls -l 
ls -la 所有
ls -l ja* 通配符查找

创建目录：

mkdir 目录名字

创建文件

touch today.c

复制文件

cp src.txt dest.txt
cp -i src.txt dest.txt 询问
cp src.txt . 复制到当前目录
cp -R 目录 dest 复制整个目录到指定路径
cp c_?1 ../  通配符复制到上级目录

删除文件

rm file.txt

删除目录

rm -rf 目录

查看文件类型：

file src.txt

查看文件内容：

cat src.txt
cat -n src.txt 显示行号

查看文件最后10行

tail -n 10 src.txt

查看文件最开始的10行

head -n 10 src.txt

安装与卸载软件

sudo apt-get install vim-gtk
sudo apt remove vim

创建用户

useradd -m jack 创建用户的同时，创建了home目录
userdel -r jack 删除用户

切换用户 

su jack

修改用户密码

sudo passwd 123456abc

退出

exit

创建组

groupadd androidgroup

分配用户到组

usermod -G androidgroup jack

文件权限

r读 w写 x执行
drwxr-xr-x
d rwx r-x r-x (d为目录，后面三个为一组，为一个二进制数)
d目录
这里用1,2,3表示上面三组数据的权限
1.文件所属用户具备的权限（root对该文件具备读写执行权限）
2.文件所属用户的所属组具备的权限（读、执行）
3.系统的其他用户具备的权限（读、执行）

rwx必须是固定顺序

权限      二进制     八进制
---         000         0
--x         001         1
-w-         010         2
-wx         011         3
r--         100         4
r-x         101         5
rw-         110         6
rwx         111         7

修改文件权限

chmod 644 file

给用户加上执行权限

chmod u+x file

改变创建目录的默认权限：

创建出来的权限就是 777 - 022 = 755权限
umask 022（默认是026）

改变文件的所属

更改用户，使file文件在user用户下
chown user file
更改用户组
chown user.group file
chown .group file

每一个用户的Hmoe目录都会有bash、bashrc文件，bash、bashrc是每一个用户登录的时候会加载的文件，里面配置的是全局的配置，例如环境变量。

vim的使用

vim有两种模式，命令模式、插入模式

命令模式中的一些基本操作：

  1. 删除一行：dd ， 删除一个字符x
  2. 撤销：u，重做：ctrl+r
  3. 进入插入模式：i
  4. 翻页：ctrl + f（上一页） ctrl + b（下一页）
  5. 复制行：yy , 粘贴：p
  6. 文本搜索： ?正则enter 搜索n或者N控制方向
  7. 保存： :w
  8. 保存退出： :wq
  9. 不保存强制退出： q!
