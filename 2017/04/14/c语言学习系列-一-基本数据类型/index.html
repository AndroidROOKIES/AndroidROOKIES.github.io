##关于c中数组的定义

c中定义数组，例如：int ids[] = { 78, 90, 23, 65, 19 };

java中定义数组有两种方式：int ids[] = { 78, 90, 23, 65, 19 }，或者int[] ids = { 78, 90, 23, 65, 19 };
