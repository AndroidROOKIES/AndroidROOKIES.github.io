C++学习系列(一)命名空间与引用

发表于 2017-05-17 | 分类于 C++

C与C++的关系

  1. C++可以与C代码进行混编，即表示C++里面可以写C，但是反过来却不可以
  2. C++是面向对象编程语言、C是面向过程的编程
  3. C++是对C的一个增强，C++有class、引用的概念，堆内存的分配释放除了C语言的malloc、free，还有new、delete关键字

C++的命名空间

C++的命名空间

命名空间，也叫名字空间，类似于Java中包（归类）。当项目比较大的时候，用于区分不同人写的代码、不同库的代码。不同命名空间下面，函数名、变量名、类名等都可以重复。

举例：

#include <iostream>

//标准命名空间(包含很多标准的定义)
//standard
//命名空间类似于Java中包（归类）    
using namespace std;

void main(){
    //运算符重载
    //std::cout << "this is c plus plus" << std::endl;
    cout << "this is c plus plus" << endl;
    system("pause");
}

通过using namespace来使用std这个命名空间。也可以通过std::的方式来用命名空间，::叫做访问修饰符。

在C++中用cout来输出，它是头文件iostream的输出函数，由于C++可以与C代码进行混编也可以用printf()输出，其中<<是运算符重载，endl是换行的意思。

自定义命名空间

#include <iostream>

//标准命名空间
using namespace std;

//自定义命名空间
namespace NSP_A{
    int a = 9;
}

namespace NSP_B{
    int a = 12;
    //命名空间嵌套
    namespace NSP_C{
        int c = 90;        
    }
}

void main(){
    //使用命名空间
    //::访问修饰符
    cout << NSP_A::a << endl;
    cout << NSP_B::a << endl;

    cout << NSP_B::NSP_C::c << endl;    
    system("pause");
}

C++命名空间中的结构体

说明一下：结构体不能被继承，类可以被继承

include <iostream>

//自定义命名空间
namespace NSP_A{
    int a = 9;
    struct Teacher{
        char name[20];
        int age;
    };
    struct Student{
        char name[20];
        int age;
    };

}

void main(){
    //使用命名空间中的结构体
    using NSP_A::Student;
    Student t;
    t.age = 90;

    system("pause");
}

注意：C++中通过using使用了命名空间中的结构体，在使用结构体的时候，struct关键字可以省略。

C++中的结构体

//结构体

struct MyTeacher{
public:
    char name[20];
    int age;
public:
    void say(){
        cout << this->age << "岁" << endl;
    }
};
void main(){
    MyTeacher t1;
    t1.age = 10;
    t1.say();
    system("pause");
}

C++是C的增强，C++中的结构体可以有访问修饰符，一个修饰符可以修饰多个变量或者函数。

C++中的结构体与Java中的类非常类似，在C语言中只能用函数指针当作成员函数，在C++中直接类似Java中的方法当做函数。

C++中的结构体有一个this指针，指向自身。

C++中的类（class）

#define PI 3.14
//圆
class MyCircle{
//属性（共用权限访问修饰符）
private:
    double r;
    double s;
public:
    void setR(double r){
        this->r = r;
    }
    //获取面积
    double getS(){
        return PI * r * r;
    }
};

void main(){
    MyCircle c1;
    c1.setR(4);

    cout << "圆的面积：" << c1.getS() << endl;

    system("pause");
}

与Java类似，有一个tihs指针指向自身。

类的定义末尾需要有分号。

C++中布尔类型

C++中有布尔类型bool，C语言没有

void main(){
    //bool isSingle = true;
    bool isSingle = 17;
    //false -17

    if (isSingle){
        cout << "单身" << endl;
        cout << sizeof(bool) << endl;
    }
    else{
        cout << "有对象" << endl;
    }

    int a = 10, b = 20;
    //三目预算符，C++中可以左值，Java中不可以
    ((a > b) ? a : b) = 30;
    cout << b << endl;

    system("pause");
}

C++中的引用

变量名相当于内存的“门牌号”，人为地取名字，因此可以有多个别名，而这种别名就是引用。

引用的主要作用：作为函数的参数或者返回值，代替指针，使得程序可读性加强。

void main(){
    //变量名-门牌号(内存空间0x00001的别名，可不可以有多个名字？)
    int a = 10;
    //b就这个内存空间另外一个别名\
    //“&”表示C++中的引用，与取地址符要区分开
    int &b = a;    
    cout << b << endl;

    system("pause");
}
